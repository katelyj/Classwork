Cascading Style Sheets (CSS)
	- Created to separate the presentation of an html/xml page and its contents
	- Basic syntax:
		- PROPERTY: VALUE;
		  color: lightsteelblue; (text color, always)
		  text-align: center;
		  border: 10px; (no space between number and unit)
	- 3 ways to incorporate CSS into a page: inline, internal style sheet, external style sheet
		- Inline CSS
			- Add CSS code to the style attribute of a tag
			- Probably used the least
			- <TAG style="CSS CODE">
			- <p style="color: green; font-size: 2em;"> ... </p>
