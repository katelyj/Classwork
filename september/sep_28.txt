When a web browser goes to a web server
- Called a "request"
- Requesting that website, respond with something
- Response codes:
	- 200 --> ok!
	- 301 --> redirect
	- 404 --> file not found
	- 405 --> method not allowed
	- 503 --> service unavailable
- print request.headers
	- Provides web browser identification
- Regular web requests: GET requests
