virtualenv
- Controls how python works/runs on a computer
- NOT an actual virtual environment
- Just makes that version of Python work everywhere!
- can be reused

$ pip install flask
$ pip install -update

jinja2
- Allows you to use html files in python code, rather than writing out html in python
- Takes it out of actual python code
- Template files must be stored in a templates/ directory at same level as Flask app file
- render_template(<FILE>, <VAR1>, <VAR2>, ...)
	- Flask function to take teplate file and return it as a string
- Put at top of file: import render_template
- For restarting server automatically: app.debug = True

- {{ message }} --> variable in html template, passed in python file
